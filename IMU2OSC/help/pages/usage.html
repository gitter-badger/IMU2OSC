IMU2OSC Usage

Overview

IMU2OSC is a utility for receiving data from an x-IMU unit and converting it to the Open Sound Control (OSC) protocol.

Requirements

In order to use this utility, the third-party FTDI Virtual COM port driver must be installed. This will enable the x-IMU unit to show up as a standard serial device

Main Window

➊Target IP address and Port
➋Serial Port selection (including bluetooth) and capture toggle
➌Connection status area
➍Incoming data display area

➊Target IP address and Port — The “IP” and “Port” text boxes allow the target IP address (IP version 4) and UDP Port to be set. These values can be changed on the fly.

➋Serial Port selection (including bluetooth) and capture toggle — The Serial Port dropdown menu can be used to select the serial port that IMU2OSC is capturing from. For Bluetooth connections, the port path will be of the form /dev/cu.x-IMU-XXXX-SPP where XXXX is the ID of your device. For USB connections, the port path will be of the form /dev/cu.usbserial-XXXXXXXX.

The Capture Toggle is used to toggle data capture from the x-IMU unit. When the checbox is checked, data will captured and displayed at in the Incoming Data display area. This checkbox also toggles the sending of value over OSC — if the checkbox is checked, values will be sent. The display rate and the OSC send rate are independent. OSC values will be sent as soon as they are emitted from the device. Values are displayed in the Data Display area at a much lower rate.

➌Connection status area — The Connection Status Area displays status messages about the connection status of the device including notification of connection attempts. If IMU2OSC has successfully connected to the device, a message of the form Connected to device with ID 0xXXXX will be show where XXXX is the ID of the device

➍Incoming data display area — The Incoming Data Display Area displays real-time values emitted from the x-IMU device. These correspond to readings from the device as follows:

  • ➠Euler — This gives the x, y and z components (in degrees) of the Euler angle representing the orientation of the x-IMU
  • ➠Gyro — This gives the calibrated values for angular velocities (in degrees per second) around the x, y and z axes of the x-IMU
  • ➠Mag — This gives the calibrated values for magnetic flux (in Gauss) along the x, y and z axes of the x-IMU
  • ➠Accel — This gives the calibrated values for acceleration (in ‘g’) along the x, y and z axes of the x-IMU

Troubleshooting

❗My device isn't showing up in the Serial Port selection menu

If your device isn't showing up try the following

  • Ensure thee battery is charged on the device
  • Plug the device into a different USB port
  • You have correctly paired the device via bluetooth
  • You have power cycled the device (turn off and on again)
